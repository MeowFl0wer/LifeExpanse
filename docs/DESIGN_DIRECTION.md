# LifeExpanse Design Direction

## Design goal

LifeExpanse should feel:

- Bright
- Clean
- Calm
- Clear
- Spacious
- Suitable for long-term reading and personal recording

## Visual principles

- Use a light theme as the primary theme.
- Prefer whitespace and typography over decorative cards.
- Avoid generic enterprise dashboard styling.
- Avoid glassmorphism.
- Avoid large gradients.
- Avoid excessive shadows.
- Avoid excessive border radius.
- Avoid overly colorful interfaces.
- Avoid large decorative illustrations.
- Do not use literal grassland or wilderness imagery.

## Suggested palette

- Page background: #F7F9FC
- Primary surface: #FFFFFF
- Primary text: #182230
- Secondary text: #667085
- Border: #E5EAF0
- Primary accent: #4F86F7
- Accent hover: #376FD8
- Subtle accent background: #EEF4FF
- Error: #D92D20
- Success: #169B62

## Typography

- Use system fonts with Chinese and English support.
- Suggested stack:
  Inter, "Noto Sans SC", "PingFang SC", system-ui, sans-serif
- Avoid excessive bold text.
- Long-form content should have comfortable line height.
- Article reading width: 720–780px.
- Main page maximum width: 1160–1200px.

## Components

- Standard border radius: 8–12px.
- Buttons should be compact and clear.
- Shadows should be very subtle or absent.
- Use borders and spacing to separate content.
- Do not wrap every section in a card.
- Saved content must remain read-only until Edit is explicitly clicked.

## Responsive behavior

- Desktop: 1440px
- Tablet: 768px
- Mobile: 390px
- Mobile layouts must be redesigned, not merely compressed desktop layouts.
